# Dynamic-AI-Intelligent-Chatbot
An AI-powered chatbot built using Flask, Groq Llama 3, and Natural Language Processing (NLP) with Context Memory, Intent Recognition, Sentiment Analysis, Named Entity Recognition, and an Analytics Dashboard.
An AI-powered conversational chatbot developed using **Flask**, **Groq Llama 3**, and **Natural Language Processing (NLP)**. The chatbot understands user intent, detects sentiment, extracts named entities, remembers conversation context, and generates intelligent responses through Generative AI.

## вҡҷпёҸ Installation

### Clone the repository

```bash
git clone https://github.com/<your-username>/AI-Intelligent-Chatbot.git
```

### Move into the project folder

```bash
cd AI-Intelligent-Chatbot
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Create a `.env` file

```text
GROQ_API_KEY=your_api_key_here
```

### Run the application

```bash
python app.py
```
